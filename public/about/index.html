What we are

npmx is a great UX/DX for the npm package registry and tooling. We strive to provide a fast, modern interface for exploring packages, with features like dark mode, keyboard navigation, code browsing, and connections to alternative registries like JSR.

We also aim to provide a great admin UI for managing your packages, teams, and organizations — all from the browser, powered by your local npm CLI.

What we're not

  • —Not a package manager. There are already people building really cool package managers.
  • —Not a registry. We don't host packages. We're just a fast, modern way to browse them.
